## Quick Start

See [`deployment.md`](deployment.md) for full instructions. Requirements: Docker Desktop.

```bash
git clone https://github.com/cp3402-students/project-2026-jcua-team7.git
cd project-2026-jcua-team7
cp .env.example .env
docker compose up -d
```

Then open http://localhost:8080 to complete the WordPress install and activate the **Tennis Blast** theme.
